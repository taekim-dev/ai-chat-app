# Button Component

The Button component is a versatile UI element that provides various styles and states for user interactions.

## Features

- Multiple variants (primary, secondary, outline, ghost, neutral, success, error, info)
- Different sizes (sm, md, lg)
- Icon support
- Loading state
- Disabled state
- Full width option

## Usage

```vue
<template>
  <Button variant="primary" size="md">
    Click me
  </Button>
</template>
```
