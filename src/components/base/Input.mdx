# Input Component

The Input component is a form control that allows users to enter text, numbers, or other data.

## Features

- Multiple input types (text, password, email, number)
- Different sizes (sm, md, lg)
- Label support
- Error state
- Disabled state
- Required state
- Full width option
- Placeholder support

## Usage

```vue
<template>
  <Input
    v-model="value"
    label="Username"
    placeholder="Enter your username"
  />
</template>
```
